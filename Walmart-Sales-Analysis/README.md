# Walmart Sales Analysis

## Business Problem
Walmart wants to identify which stores generate the highest weekly sales and how holiday weeks impact revenue.

## Tools Used
- SQL
- PostgreSQL
- Power BI
- Excel

## Dataset
Walmart weekly sales dataset from Kaggle.

## Key Questions
- Which stores perform best?
- Do holidays increase weekly sales?
- What are the yearly trends?
- Which store has the highest average sales?

## Key Insights
- Store 20 generated the highest total sales
- Holiday weeks showed stronger spikes
- Some stores showed seasonal patterns
